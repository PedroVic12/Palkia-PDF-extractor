No Notepad++, vá novamente para Linguagem -> Linguagem Definida pelo Usuário -> Defina sua linguagem....



Na janela, selecione a linguagem Organon Script (Teste).



Vá para o separador Keywords Lists.



À direita, você verá caixas para Grupo 2, Grupo 3, etc. Faremos o seguinte:



Para os Comandos de Execução (Roxo):



Clique na caixa Grupo 2.



Copie e cole a seguinte lista lá dentro: NEWTON EXLF SDPF CPF PFCTG SOLVE



Clique no botão Styler ao lado.



Escolha uma cor roxa/magenta, marque a caixa Negrito e clique em OK.



Para o Controlo de Fluxo (Vermelho Escuro):



Clique na caixa Grupo 3.



Copie e cole: LOOP ENDLOOP IF ENDIF



Clique no botão Styler, escolha uma cor vermelho escuro, marque Negrito e clique em OK.



Para Comandos de E/S (Verde-azulado):



Clique na caixa Grupo 4.



Copie e cole: REPORT EXPORT IMPORT MSG



Clique no botão Styler, escolha uma cor verde-azulado e clique em OK.



Para os Parâmetros (Laranja Queimado):



Clique na caixa Grupo 5.



Copie e cole a lista de parâmetros: BCan Iden SE SA Ar P.CC Pmx P Bger Qmn Qmx Tp VBASE VMIN VMAX PUNIT QUNIT R X TAP MVA MVAR STAT TYPE AREA ZONE NOME TENSAO ANGULO PCARGA QCARGA VDEF VFIN PMIN PMAX QMIN QMAX



Clique no botão Styler, escolha uma cor laranja queimado e clique em OK.



Não precisa reiniciar. As alterações devem ser aplicadas imediatamente ao seu ficheiro .spt aberto.
